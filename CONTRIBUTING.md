# Contributing to 3D Sound Journey Builder

Thank you for your interest in contributing! We welcome contributions to enhance the spatial audio features, improve user interaction, and optimize AI-driven soundscape generation.

## How to Contribute

1. **Search Issues**: Check if your bug or feature request is already reported.
2. **Fork the Repository**: Create a fork of the project.
3. **Branch**: Create a new branch named `feat/your-feature` or `fix/your-patch`.
4. **Develop**:
   - Write clean JavaScript/TypeScript.
   - Maintain compatibility with standard WebAudio API behaviors.
   - Test changes locally (`npm run test` if tests are configured, or verify manually in browser).
5. **Commit**: Use Conventional Commits formatting (e.g. `feat: add binaural panner node override`, `fix: resolve sound source drag clipping`).
6. **PR**: Open a pull request against the `main` branch. Provide a clear description of changes.

## Development Setup

```bash
# Install dependencies
npm install

# Start Vite local development server
npm run dev
```
