# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-06

### Added
- **Core Spatial Audio Engine**: WebAudio API implementation supporting binaural stereo panning, distance attenuation, and multi-source spatial positioning.
- **Drag-and-Drop Soundscape Canvas**: Coordinate-based layout allowing users to position audio sources relative to the listener (center).
- **Qwen-Driven Curation**: Integration with Chinese LLMs (Qwen3.6 Plus) to analyze prompt parameters and generate structural soundscape configurations.
- **Dynamic Oscillators & Audio Nodes**: Support for custom synthesizers, volume adjustments, and dynamic sound triggers.
- **Vite Setup**: Hardware-accelerated local development and packaging workflow.
